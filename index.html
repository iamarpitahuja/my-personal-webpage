Featured Project

Distance Simulation

A open source Python app simulating the localization readings of a VEX robot. It simulates the VEX distance sensors alongside a particle filter to estimate the robot's position. Runs a standard gaussian noise model to simulate the sensor noise. Used to relocalize the robot after potential drift.

Featured Project

Echo Library

Echo-Lib is a private PROS library for VEX competition robots. It has algorithms for moving robots using Pure Pursuit, motion profiling, and PID control. It is a powerful system for developing consistent autonomous routines. Will be made open source following the end of 2026 VEX Robotics Season.
